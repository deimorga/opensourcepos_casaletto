La aplicación ofrece soporte para muchos países, y los montos en moneda se imprimen en la denominación local. En algunos países (y algunas ubicaciones) puede ser conveniente imprimir un recibo o factura en otra denominación.

Para que esto sea posible, cada venta que se vaya a imprimir en una denominación alternativa requiere que se proporcione el tipo de cambio.

Para configurar el sistema de modo que soporte una segunda moneda...

* Ve a Configuración y selecciona la pestaña de Localización.
* Marca la casilla "Usar Moneda Alternativa".
* Determina el código de idioma que indica cómo debe formatearse el número para su visualización e ingrésalo en el campo "Localización Alternativa".
* Si el código de moneda no es correcto (por ejemplo, si quieres usar Euro en lugar de Libra Esterlina) puedes proporcionar el símbolo real en el campo "Símbolo de Moneda Alternativa".

Ahora, cuando ingreses una venta, en la parte inferior de la sección de la derecha debería aparecer una casilla para "Aplicar Tipo de Cambio" y, junto a ella, el campo donde se ingresa el tipo de cambio. El último tipo de cambio usado se te proporciona automáticamente (el último tipo de cambio ingresado se guarda internamente en la tabla de configuración) y puedes reemplazarlo con el tipo de cambio que se deba usar para esta venta.

Si cierras una venta sin seleccionar "Aplicar Tipo de Cambio", puedes ir a ventas diarias, ubicar la venta, ir a "Actualizar" y cambiar el tipo de cambio a usar. Luego, al volver a la lista de documentos de venta, habrá dos "íconos de impresión". El primero imprimirá el documento en la moneda local y el segundo (el que tiene las dos flechas apuntando en direcciones diferentes) imprimirá el documento de venta usando la moneda alternativa.

El tipo de cambio, el idioma alternativo y el símbolo de moneda alternativa se guardan con cada venta, de modo que el documento de venta se pueda reimprimir con el tipo de cambio y el código de moneda con los que el sistema estaba configurado en el momento de la venta.
