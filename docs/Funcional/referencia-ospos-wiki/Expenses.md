Since version 3.2.0 there's a basic expenses module with simple logging of expenses.

Usage is as follows: click the register button and a dialog popup will appear, fill in the fields and submit the data. More info in [Complete Feature Datasheet](Complete-feature-datasheet) and [Getting Started Usage](Getting-Started-usage).
