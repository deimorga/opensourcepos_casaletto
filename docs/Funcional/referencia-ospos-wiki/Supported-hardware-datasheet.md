A standard setup is comprised of a web server, a web browser client, a barcode reader, a cash drawer and a receipt printer.

## Supported Devices

| Device | Related Information |
|--------|---------------------|
| **Server (Runtime)** | Any hardware that runs PHP, MySQL and a web server. See [Minimum Server Requirements](Minimum-Server-Requirements). |
| **Client (Usage)** | Any hardware running Firefox or a modern web browser. Works on mobile but PC/Mac is recommended. |
| **Barcode Scanners** | Any scanner will work if programmed to hit return after scanning. Recommended: [Datalogic QuickScan](https://www.amazon.com/gp/search/ref=as_li_qf_sp_sr_tl?ie=UTF8&tag=ospos-20&keywords=datalogic%20quickscan&index=aps&camp=1789&creative=9325&linkCode=ur2&linkId=4278b9783c5aaad9dc17ae38debc6f3e) - easy to configure and well supported. Mobile barcode reading not yet supported. |
| **Printers** | Depends on [browser and OS](Printing). Recommended: [Star TSP 1431IU](https://www.amazon.com/gp/search/ref=as_li_qf_sp_sr_tl?ie=UTF8&tag=ospos-20&keywords=Star%20TSP143&index=aps&camp=1789&creative=9325&linkCode=ur2&linkId=68fb044e8a64e9842477d82710e8fd70) or [Epson TM-88VI](https://amzn.to/38XX2VL). Use our referral links to support the project! |
| **Cash Drawers** | Can be handled by some printers. Cash drawers are out of scope due to the system being web based. |
| **Credit Card Reader** | Not supported |
| **Digital Signature Capture** | Not supported yet |
