Una configuración estándar está compuesta por un servidor web, un navegador web cliente, un lector de código de barras, un cajón de dinero y una impresora de recibos.

## Dispositivos compatibles

| Dispositivo | Información relacionada |
|--------|---------------------|
| **Servidor (Runtime)** | Cualquier hardware que ejecute PHP, MySQL y un servidor web. Consulta [Requisitos mínimos del servidor](Minimum-Server-Requirements). |
| **Cliente (Uso)** | Cualquier hardware que ejecute Firefox u otro navegador web moderno. Funciona en móviles, pero se recomienda PC/Mac. |
| **Escáneres de código de barras** | Cualquier escáner funcionará si está programado para enviar un retorno (enter) después de escanear. Recomendado: [Datalogic QuickScan](https://www.amazon.com/gp/search/ref=as_li_qf_sp_sr_tl?ie=UTF8&tag=ospos-20&keywords=datalogic%20quickscan&index=aps&camp=1789&creative=9325&linkCode=ur2&linkId=4278b9783c5aaad9dc17ae38debc6f3e) - fácil de configurar y con buen soporte. La lectura de códigos de barras desde móviles aún no es compatible. |
| **Impresoras** | Depende del [navegador y el SO](Printing). Recomendadas: [Star TSP 1431IU](https://www.amazon.com/gp/search/ref=as_li_qf_sp_sr_tl?ie=UTF8&tag=ospos-20&keywords=Star%20TSP143&index=aps&camp=1789&creative=9325&linkCode=ur2&linkId=68fb044e8a64e9842477d82710e8fd70) o [Epson TM-88VI](https://amzn.to/38XX2VL). ¡Usa nuestros enlaces de referido para apoyar el proyecto! |
| **Cajones de dinero** | Algunas impresoras pueden manejarlos. Los cajones de dinero están fuera del alcance del sistema, dado que este es basado en web. |
| **Lector de tarjetas de crédito** | No compatible |
| **Captura de firma digital** | Aún no compatible |
