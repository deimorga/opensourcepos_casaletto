La aplicación cuenta con una funcionalidad de "Mesa" (Table) dentro del módulo de **Venta** (Sale) para uso en restaurantes. Para la documentación de otros módulos, consulta [Guía de Uso - Primeros Pasos](Getting-Started-usage).

Lista completa de módulos en la [Hoja de Datos de Funcionalidades Completa](Complete-feature-datasheet#complete-list-of-features)

## Mesas de Restaurante

Los clientes se agrupan en "Mesas" (Tables), que actúan como etiquetas para las transacciones. Puedes vender a una mesa con o sin cliente asociado, ideal para combinaciones de reuniones.

## Cómo Usarlo

1. Primero, habilita las Mesas de Restaurante en el módulo **Oficina** (Office).
2. Aparecerá un nuevo selector en el módulo **Venta** (Sale), junto al Modo de Venta.
3. Selecciona la Mesa.
4. Carga los artículos y suspende la venta para comenzar con una nueva mesa.
5. Cuando suspendes la transacción, la mesa desaparece de tu lista de selección.
6. Para reanudar la venta de otra mesa, elige esa mesa y ve al botón de suspendidas.
7. Reanuda (unsuspend) la venta de la mesa para seguir agregando artículos y/o completar la venta.

> **Nota:** Este flujo de trabajo no es muy intuitivo, pero es funcional. Se planean mejoras futuras para hacerlo más amigable para el usuario. ¡Las contribuciones son bienvenidas!
